# 🎯 Quick Start Guide

## Installation

The application requires no installation - it's a pure web application!

1. **Open in Browser**: Simply open `index.html` in any modern web browser
2. **No Dependencies**: No need to install Node.js, npm, or any other tools
3. **No Server Needed**: Runs entirely on your local machine

## First Steps

### 1. Choose Your Difficulty
- **Easy** (🟢): Best for beginners, simpler vocabulary
- **Medium** (🟡): Intermediate level, varied topics
- **Hard** (🔴): Advanced, complex texts and technical content

### 2. Select Test Mode
- **Timed (60s)**: Standard test, 60 seconds to type as much as possible
- **Full Text**: Complete challenge, finish typing the entire passage

### 3. Start Testing
- Click **"Start Typing Test"** or press any key
- Type the passage that appears
- Watch your stats update in real-time

### 4. View Results
- Your WPM (Words Per Minute), accuracy, and stats are calculated
- New personal record? Celebrate with a special message! 🎉
- Try again immediately or change difficulty

## Understanding Your Stats

### WPM (Words Per Minute)
- Standard typing speed metric
- Formula: (Characters ÷ 5) ÷ Minutes
- Average typing: 40-60 WPM
- Professional typists: 70-100+ WPM

### Accuracy
- Percentage of correct characters
- 100% = No mistakes
- 90%+ = Excellent
- 80-90% = Good
- <80% = Practice more!

### Character Count
- Total number of correctly typed characters
- Helps track consistency

## Trouble Shooting

**Test won't start?**
- Try refreshing the page
- Ensure JavaScript is enabled
- Try a different browser

**Stats not showing?**
- Clear your browser cache
- Check that you're typing in the passage area
- Ensure your keyboard is working properly

**Personal best not saving?**
- Check if localStorage is enabled
- Try a different browser
- Clear cookies if using private mode
